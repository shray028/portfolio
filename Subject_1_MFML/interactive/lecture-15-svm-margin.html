We can calculate the total width of the street using basic geometry. We'll find the distance from a reference point (the origin (0,0)) to each sidewalk.

Distance to the Positive Sidewalk ($w^T x + b = 1$):

Let $x_0$ be the point on this line closest to the origin. The vector $x_0$ (from the origin to this point) must be perpendicular to the line. Since $w$ is also perpendicular, $x_0$ must be a scaled version of $w$. So, $x_0 = k \cdot w$ for some scaling number $k$.

We know $x_0$ is on the line, so it satisfies the line's equation: $w^T x_0 + b = 1$.

Let's substitute $x_0 = k \cdot w$ into the equation: $w^T(k \cdot w) + b = 1$.

This simplifies to $k(w^T w) + b = 1$. Since $w^T w$ is the squared length of $w$ ($||w||^2$), we have $k ||w||^2 + b = 1$.

Solving for our scaling factor: $k = (1 - b) / ||w||^2$.

The distance is the length of the vector $x_0$, which is $||x_0|| = |k| \cdot ||w||$. Plugging in k:

Distance$_+ = |(1 - b) / ||w||^2| \cdot ||w|| = |1 - b| / ||w||$.

Total Margin:

Repeating the same math for the negative sidewalk ($w^T x + b = -1$) gives its distance from the origin as $|-1 - b| / ||w||$.

The total margin is the distance between the two lines. The margin width is cleanly defined as $2/||w||$.
